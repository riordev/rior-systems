## Notes for Actual Screenshots

When capturing real screenshots from Looker Studio:

1. **Use sample data** from `sql/placeholder_data.sql`
2. **Browser:** Chrome, full screen, 1440x900 minimum
3. **Zoom:** 100% (no browser zoom)
4. **Theme:** Ensure dark theme is applied
5. **Hide UI:** Use Looker Studio "View" mode to hide editor chrome
6. **Capture each page** separately
7. **Mobile:** Use Chrome DevTools mobile emulator (iPhone 12 Pro)
8. **File naming:**
   - `executive-summary.png`
   - `product-performance.png`
   - `marketing-performance.png`
   - `alerts-signals.png`
   - `mobile-view.png`

Store screenshots in `/assets/screenshots/` directory.
