## 4. DATA STORAGE AND SECURITY

**Storage:**
- All data stored in Google Cloud Platform (US-Central region)
- Encrypted at rest using AES-256
- Encrypted in transit using TLS 1.3

**Access Controls:**
- Role-based access control (RBAC)
- Multi-factor authentication for all staff
- Access logging and monitoring
- Regular security audits

**Data Retention:**
- Active client data: Retained for duration of service plus 90 days
- Deleted data: Securely wiped within 30 days of deletion request
- Backups: Encrypted and retained for 30 days
