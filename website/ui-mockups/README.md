# Rior Systems — UI Mockup Variations

## Overview
Four distinct design directions for the Profit Intelligence dashboard, inspired by different aesthetics and user needs.

---

## Vibe 1: "Terminal" (Dark, High-Contrast, Data-Dense)
**Inspiration:** Bloomberg terminal, Linear.app, Vercel analytics
**Colors:** 
- Background: `#0a0a0a` (pure black)
- Surface: `#141414` (elevated cards)
- Border: `#262626` (subtle separation)
- Text Primary: `#fafafa` (white)
- Text Secondary: `#a1a1aa` (gray)
- Accent: `#22c55e` (profit green) / `#ef4444` (loss red)

**Key Features:**
- Monospace font for numbers (JetBrains Mono)
- Grid layout, data-dense
- Sparklines everywhere
- Keyboard shortcuts visible
- No wasted space

**Best For:** Power users, agencies, high-volume merchants

---

## Vibe 2: "Amber" (Warm, Approachable, Diary-like)
**Inspiration:** tomsvirtualdiary.com, personal journaling apps
**Colors:**
- Background: `#faf9f6` (warm white)
- Surface: `#ffffff` (pure white cards)
- Border: `#e7e5e4` (warm gray)
- Text Primary: `#292524` (warm black)
- Text Secondary: `#78716c` (taupe)
- Accent: `#d97706` (amber) / `#059669` (soft green)

**Key Features:**
- Serif headings (Crimson Pro) for warmth
- Sans-serif body (Inter)
- Generous whitespace
- Photo/avatar integration
- Timeline-style activity feed

**Best For:** Founder-led brands, personal touch, storytelling

---

## Vibe 3: "Glass" (Modern, Floating, Premium)
**Inspiration:** Apple design, macOS Big Sur, modern SaaS
**Colors:**
- Background: `#000000` with blur
- Surface: `rgba(255,255,255,0.05)` (glassmorphism)
- Border: `rgba(255,255,255,0.1)`
- Text Primary: `#ffffff`
- Text Secondary: `rgba(255,255,255,0.6)`
- Accent: `#3b82f6` (blue) gradients

**Key Features:**
- Backdrop blur effects
- Floating cards
- Gradient accents
- Smooth animations
- iOS-style widgets

**Best For:** Premium positioning, tech-forward brands

---

## Vibe 4: "Editorial" (Bold, Magazine-like, Content-Forward)
**Inspiration:** The Economist, Notion, high-end publications
**Colors:**
- Background: `#ffffff`
- Surface: `#f8fafc` (slate-50)
- Border: `#e2e8f0` (slate-200)
- Text Primary: `#0f172a` (slate-900)
- Text Secondary: `#64748b` (slate-500)
- Accent: `#0ea5e9` (sky blue) / `#f59e0b` (amber)

**Key Features:**
- Large typography
- Pull quotes for insights
- Magazine-style layouts
- Strong hierarchy
- Content blocks

**Best For:** Content marketing, educational positioning

---

## Recommendation

For **Rior Systems**, I recommend a **hybrid approach**:
- **Default:** "Terminal" aesthetic (positions you as serious/data-driven)
- **Toggle:** "Amber" mode (for founders who want warmth/approachability)
- **Accent color:** User-configurable (match their brand)

This gives the technical credibility needed for $20k+ ad spend clients while maintaining approachability for founder-led brands.
